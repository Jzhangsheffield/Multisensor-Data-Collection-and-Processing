这里面包含了5种传感器：D435i, X4m300, Vlp-16, Evk4, EMG armband 的数据采集脚本。还在逐步完善中。部分脚本需要安装对应的SDK后运行，其中
D435i: https://pypi.org/project/pyrealsense2/
X4m300: https://github.com/novelda/Legacy-SW
Vlp-16: 需要在linux中使用Ros2 运行.
Evk4: https://docs.prophesee.ai/4.6.2/index.html
EMG armband: 使用Mindrove 软件进行录制.

传感器的启动需要时间，在采集数据是请留出启动和关闭的时间。
